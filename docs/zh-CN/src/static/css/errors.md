# errors.css

> 📅 最后更新日期: 2026/06/28

负责“错误日志”页签下的搜索框、筛选器、数据表格及分页控件的样式定义。

## 搜索与筛选 (`.filter-container`)

- **布局**: 采用 `flex` 布局，左侧显示标题，右侧显示搜索输入框和节点下拉筛选器。
- **搜索框 (`.error-search-input`)**:
  - 具有平滑的过渡效果（`transition: all 0.2s`）。
  - **焦点态**: 获得焦点时，边框颜色加深，背景色略微改变以提供视觉反馈。

## 数据表格 (`#errors-table`)

- **结构**: 标准 HTML 表格，但在移动端/窄屏下会自动转换为卡片流布局。
- **单元格样式**:
  - `.error-id`: 使用灰色调显示 ID，降低视觉噪音。
  - `.error-cell`: 专门用于展示错误 repr，使用等宽字体（`monospace`）和红色调（`Crimson`）。为了防止超长错误信息撑破布局，设置了 `max-width: 40ch` 并使用省略号（`ellipsis`）截断。
  - `.retry-link`: 错误行中的“重试”操作链接，`--cornflower-600` 文字色，悬停或聚焦时下划线加深，暗色模式下使用 `--cornflower-300`。
  - `.retry-disabled`: 不可重试时的占位文字，`--carbon-400` 灰字，无交互反馈，暗色模式下使用 `--carbon-500`。

## 分页控件 (`.pager-container`)

- **页码链接 (`.pager-link`)**: 支持悬停高亮。
- **翻页按钮 (`.pager-btn`)**:
  - 包含“上一页”和“下一页”图标按钮。
  - **禁用态**: 当处于第一页或最后一页时，按钮变为灰色且不可点击（`cursor: not-allowed`）。

## 响应式设计

- **表格转卡片**: 在宽度小于 `2048px` 时，表格 `thead` 隐藏，每一行 `tr` 变为一个独立的带圆角边框的卡片块。
- **伪元素标签**: 通过 `::before { content: attr(data-label) }` 将表格列名注入到卡片内部，使移动端用户依然能识别数据含义。
- **错误展开**: 在卡片模式下，`.error-cell` 的截断限制被移除（`white-space: normal`），改为自动换行以展示更多细节。
