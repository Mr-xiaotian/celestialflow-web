# i18n.ts

> 📅 最后更新日期: 2026/06/22

国际化模块，负责多语言（中、英、日）翻译数据的维护、当前语言状态管理以及 DOM 文本的自动更新。

## 类型定义

### `Lang`
支持的语言标识：`"zh-CN"` | `"en"` | `"ja"`。

## 全局变量

| 变量 | 类型 | 说明 |
|------|------|------|
| `currentLang` | `Lang` | 当前选中的语言标识，默认为 `"zh-CN"` |
| `translations` | `Record<Lang, Record<string, string>>` | 包含所有语言的翻译字典 |

## 核心函数

### `setLang(lang)`

切换全局语言设置。

- **参数**: `lang: Lang` - 目标语言标识。
- **行为**: 更新 `currentLang` 并同步修改 `<html>` 标签的 `lang` 属性。

---

### `t(key, ...args)`

根据翻译键获取当前语言的文本，支持简单的占位符替换。

- **参数**:
  - `key: string` - 翻译键名。
  - `...args: string[]` - 用于替换 `{0}`, `{1}`... 的参数列表。
- **返回**: 翻译后的字符串。如果找不到对应的键，则返回键名本身。
- **回退机制**: 如果当前语言下没有该键，会尝试回退到 `zh-CN`。

---

### `applyI18nDOM()`

扫描全页带有特定 `data-i18n` 属性的元素并应用翻译。

**支持的属性：**
- `data-i18n`: 替换 `textContent`。
- `data-i18n-placeholder`: 替换 `placeholder`。
- `data-i18n-title`: 替换 `title` 属性。
- `data-i18n-aria-label`: 替换 `aria-label` 属性。

此外，该函数还会同步更新 `document.title`。

## 翻译键分组

- `app.*`: 应用全局文本（标题等）
- `settings.*`: 设置面板相关
- `tab.*`: 标签页名称
- `card.*`: 卡片标题
- `summary.*`: 汇总数据标签
- `errors.*`: 错误日志相关
- `injection.*`: 任务注入相关
- `status.*`: 节点状态相关
- `chart.*`: 图表相关
- `interval.*`: 时间间隔选项

## 使用示例

### t() / setLang() / applyI18nDOM() 的使用示例

以下示例展示国际化模块的三个核心函数在浏览器控制台中的用法：

```typescript
// 1. t()：获取当前语言的翻译文本
console.log(t("app.title"));       // "任务图监控系统" (当前 zh-CN)
console.log(t("settings.saveSuccess")); // "设置已保存"

// 2. t() 支持占位符替换
// 翻译键 "injection.successBatch": "已成功提交 {0} 个节点草稿"
console.log(t("injection.successBatch", "3"));
// "已成功提交 3 个节点草稿"

// 3. t() 的降级机制
// 如果当前语言（如 ja）没有某个键，回退到 zh-CN
// 如果 zh-CN 也没有，则返回键名本身
console.log(t("nonexistent.key"));  // "nonexistent.key"

// 4. setLang()：切换全局语言
setLang("en");
console.log(currentLang);           // "en"
console.log(document.documentElement.lang); // "en"
console.log(t("app.title"));       // "Task Graph Monitor"

setLang("ja");
console.log(t("app.title"));       // "タスクグラフモニター"

// 5. applyI18nDOM()：扫描全页并更新翻译
// 假设 HTML 中有：
//   <h1 data-i18n="app.title"></h1>
//   <input data-i18n-placeholder="errors.searchPlaceholder">
//   <div data-i18n-title="settings.tooltip"></div>
//
// 调用 applyI18nDOM() 后：
//   <h1>任务图监控系统</h1>
//   <input placeholder="搜索错误或任务">
//   <div title="设置"></div>

// 6. 完成的语言切换流程
function switchLanguage(lang: Lang) {
    setLang(lang);                    // 更新 currentLang 和 <html lang>
    applyI18nDOM();                   // 更新所有 data-i18n 元素
    document.title = t("app.title"); // 更新页面标题
}

// 切换为英文
switchLanguage("en");

// 切换为中文
switchLanguage("zh-CN");

// 7. 检查某个翻译键是否存在
function hasTranslation(key: string, lang?: Lang): boolean {
    const targetLang = lang || currentLang;
    return !!translations[targetLang]?.[key];
}

console.log(hasTranslation("app.title"));                 // true
console.log(hasTranslation("app.title", "en"));           // true
console.log(hasTranslation("app.title", "ja"));           // true
console.log(hasTranslation("nonexistent.key", "zh-CN"));  // false
```
