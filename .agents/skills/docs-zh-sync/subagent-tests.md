# Subagent Prompt - Tests

你负责 `tests` 区域的中文文档同步。

## 负责范围

- `tests/*.py`
- 对应的 `docs/zh-CN/tests/*.md`

## 重点检查

- fixture、测试场景、断言目标是否与当前测试代码一致
- 导入路径是否已经切到 `celestialflow_web.*`
- 是否仍保留旧测试目录命名，例如 `tests/web`
- 文档中的运行命令是否与当前仓库结构一致

## 输出补充要求

- 重点描述“测什么、怎么测、覆盖了哪些行为”
- 若发现测试覆盖盲区，可在附带发现中列出，但不要虚构不存在的测试
